# Go-chats
## Стек:
- Go 1.15
- Socket.io

### Запуск:
~~~
go build -v ./app/cmd
go run ./app/cmd
~~~
### Колличество одновременных соединений: 
В params-local.json укажите необходимое колличество соединений(по умолчанию 65535)
Для подняния лимита файловых дескрипторов в текущей сессии:
~~~
ulimit -n 200000
~~~
